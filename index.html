PSN — What this is

A collaborative documentation hub for Project Solar Node. This site stores: build logs, BOMs, wiring diagrams, code snippets, and how‑tos we want to share within the PSN community and with future contributors.

Why this approach

  • Single-source docs (Markdown) that convert to a static website.
  • Lightweight, fast, and easy to host (GitHub Pages, Netlify, Vercel).
  • Designed to be machine-readable so ChatGPT can ingest and reference it later.

BOM (Shelf1BOM)

We'll store a CSV/Google Sheet with columns: Part name, Qty, Link, Vendor, Cost, Date ordered, Install location, Notes.

Example quick CSV header: part,qty,link,vendor,cost,date,location,notes

Tip: keep the BOM in the repo under /bom/shelf1bom.csv so it's versioned and easy to export.

How you can share docs with me (the assistant)

  • Upload files directly to this chat — easiest for quick excerpts, images, diagrams.
  • Connect a Google Drive or Dropbox and share the folder — best for ongoing sync and lots of assets.
  • Host the repo on GitHub and give me the link or push updates — I can generate PR-ready content that you paste into the repo.

I recommend starting with direct uploads for Phase 2 photos + diagrams and a single GitHub repo named psn-site. I'll create Markdown pages from your content and push a deployable static site build you can host for free.
